## Features

- **Interactive README Generation**: Create professional README files through an intuitive web interface
- **Real-time Preview**: See your README markdown rendered in real-time as you build it
- **Customizable Sections**: Add, remove, and reorder common README sections like installation, usage, and contributing
- **Template Library**: Choose from pre-built templates for different project types
- **Export Options**: Download your generated README as a markdown file
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **No Registration Required**: Generate README files instantly without creating an account

## Installation

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Setup

1. Clone the repository:
```bash
git clone https://github.com/adivrskic/readme-generator.git
cd readme-generator
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

### Production Build

To create a production build:
```bash
npm run build
```

## Usage

### Basic Usage

1. **Open the Application**: Navigate to the README Generator in your web browser
2. **Choose a Template**: Select from available README templates or start from scratch
3. **Fill in Project Details**: Add your project name, description, and other basic information
4. **Add Sections**: Use the interface to add sections like installation, usage, contributing guidelines, etc.
5. **Customize Content**: Fill in the content for each section using the provided forms
6. **Preview**: Review your README in the live preview panel
7. **Export**: Download the generated README.md file to your project directory

### Example Workflow

```
Project Name: My Awesome App
Description: A web application that does amazing things
Technology: React, Node.js
License: MIT

Generated sections:
✓ Installation instructions
✓ Usage examples  
✓ API documentation
✓ Contributing guidelines
✓ License information
```

### Advanced Features

- **Custom Badges**: Add status badges for build, coverage, and version information
- **Code Examples**: Include syntax-highlighted code snippets in your README
- **Image Integration**: Add screenshots and diagrams to enhance documentation
- **Table Generation**: Create formatted tables for API references or feature comparisons
